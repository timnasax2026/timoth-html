## 🛠️ How to Deploy (Step-by-Step)

### Step 1: Pairing
Get your **Session ID** first. You cannot run the bot without it.
* Open the [Pairing Site](https://pairing-web-su41.onrender.com/).
* Enter your WhatsApp number with country code (e.g., `+1...`).
* Follow the instructions to link the device and save the Session ID sent to your chat.

### Step 2: Fork the Bot
Make sure to fork the main bot engine here:
* [Official Bot Repo](https://github.com/stanytz378/IAMLEGEND)

### Step 3: Deployment
Use our dashboard to select one of the **15 Cloud Platforms** available. 
* **Render** & **Koyeb** are highly recommended for beginners.
* **Railway** & **Heroku** are great for stability.
